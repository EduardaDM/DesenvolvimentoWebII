# Desenvolvimento-Web-II
Repositório para postagem de trabalhos da matéria de Desenvolvimento Web II, do quarto semestre de Ciência da Computação.
Trabalhos realizados: Swagger, SOAP, API REST e diagramas, de componentes e de implantação. 
Os trabalhos foram feitos de forma individual, com o metodo "sala invertida". Todos os trabalhos foram feitos na IDE Eclipse com a biblioteca Web Servie disponibilizada pela professora.
